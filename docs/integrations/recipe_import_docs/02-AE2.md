# Applied Energistics 2 / AE2

## Adapter IDs

- `appliedenergistics2:grinder`：AE2 石英研磨机，读取 `AEApi.instance().registries().grinder().getRecipes()`。
- `appliedenergistics2:inscriber`：AE2 压印器，读取 `AEApi.instance().registries().inscriber().getRecipes()`。

## 示例

```zenscript
import mods.modularmachinery.RecipeAdapterBuilder;
import mods.modularmachinery.RecipeModifierBuilder;

RecipeAdapterBuilder.create("mm_ae2_grinder", "appliedenergistics2:grinder")
    .addModifier(RecipeModifierBuilder.create("modularmachinery:item", "output", 2.0, 1, true).build())
    .build();

RecipeAdapterBuilder.create("mm_ae2_inscriber", "appliedenergistics2:inscriber").build();
```

## 说明

AE2 UEL 的公开机器配方 registry 只稳定暴露研磨机与压印器。充能器 API 是充电倍率，不是可枚举机器配方；AE2Stuff 等 addon 若复用 AE2 核心压印/研磨表，会随这两个 adapter 一起反映。
