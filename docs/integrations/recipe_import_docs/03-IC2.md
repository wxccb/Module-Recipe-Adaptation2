# IndustrialCraft 2 / IC2

## Adapter IDs

- `ic2:macerator`：打粉机
- `ic2:extractor`：提取机
- `ic2:compressor`：压缩机
- `ic2:centrifuge`：热能离心机
- `ic2:blockcutter`：方块切割机
- `ic2:blastfurnace`：高炉
- `ic2:recycler`：回收机
- `ic2:metalformer_extruding`：金属成型机-挤压
- `ic2:metalformer_cutting`：金属成型机-切割
- `ic2:metalformer_rolling`：金属成型机-轧制
- `ic2:ore_washing`：洗矿机
- `ic2:electric_furnace`：电炉
- `ic2:canner_bottle`：装罐机-灌装/装瓶
- `ic2:canner_enrich`：装罐机-流体富集
- `ic2:electrolyzer`：电解机
- `ic2:fermenter`：发酵机
- `ic2:matter_amplifier`：UU 物质放大物
- `ic2:scrapbox_drops`：废料盒概率掉落

## 示例

```zenscript
import mods.modularmachinery.RecipeAdapterBuilder;
import mods.modularmachinery.RecipeModifierBuilder;

RecipeAdapterBuilder.create("mm_compressor", "ic2:compressor")
    .addModifier(RecipeModifierBuilder.create("modularmachinery:item", "input", 2.0, 1, false).build())
    .addModifier(RecipeModifierBuilder.create("modularmachinery:item", "output", 2.0, 1, false).build())
    .build();

RecipeAdapterBuilder.create("mm_scrapbox", "ic2:scrapbox_drops")
    .addModifier(RecipeModifierBuilder.create("modularmachinery:item", "output", 2.0, 1, true).build())
    .build();
```

## 说明

IC2 使用 `ic2.api.recipe.Recipes` 的最终 registry。洗矿机会自动加入水输入；废料盒掉落按概率输出导入。
